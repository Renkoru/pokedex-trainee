Immutable Data Structures

Do not mutate the state object!

  • JavaScript's primitive data types (Number, String, Boolean, Undefined, and Null) are already immutable.
  • Objects, arrays, and functions are mutable.

How to implement immutability

          const newState = Object.assign({}, state, { foo: 123 });

Middlewares

  • Middleware is some code you can put between the framework receiving a request, and the framework generating a response.
  • Redux middleware solves different problems than Express or Koa middleware, but in a conceptually similar way. It provides a third-party extension point between dispatching an action, and the moment it reaches the reducer.
  • (Restriction) Last middleware in a chain sould return a plain object
